# arr2obj

Converts an array to object, with customization of key and value.

### Examples

Array of basic types:
```js
arrayToObject(['a', 'b', 'c', 'd'])
// Returns:
// { a: 'a', b: 'b', c: 'c', d: 'd' }
```
